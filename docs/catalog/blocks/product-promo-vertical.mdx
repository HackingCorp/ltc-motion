# Product Promo Vertical

Vertical 9:16 e-commerce product promo with product card, price badge, and WhatsApp order CTA — built for WhatsApp Status, Reels, and TikTok. Fork addition from HackingCorp/ltc-motion.

`ecommerce` `product` `vertical` `whatsapp` `promo` `social`

## Install

<CodeGroup>

```bash Terminal
npx hyperframes add product-promo-vertical
```

</CodeGroup>

## Details

| Property | Value |
| --- | --- |
| Type | Block |
| Dimensions | 1080×1920 |
| Duration | 7s |

## Files

| File | Target | Type |
| --- | --- | --- |
| `product-promo-vertical.html` | `compositions/product-promo-vertical.html` | hyperframes:composition |
| `assets/product.svg` | `assets/product.svg` | hyperframes:asset |

## Usage

After installing, add the block to your host composition:

```html
<div data-composition-id="product-promo-vertical" data-composition-src="compositions/product-promo-vertical.html" data-start="0" data-duration="7" data-track-index="1" data-width="1080" data-height="1920"></div>
```
