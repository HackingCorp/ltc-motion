Global Motion Notes

Canvas Halftone System
Background is a canvas-rendered halftone field (from halftone-motion-examples.html), NOT CSS radial-gradient dots. Key components: drawHalftone() with fbm domain warping, gradient masses (3 elliptical color fields that drift), dot-level energy/color from shaderField(), and paintShaderWash() soft color overlay in screen blend mode. Spacing = 5.15 / density, base radius = 1.02. The field is one continuous warped surface — dots are texture, not independent particles.
Transition Modes (per beat)
FIELD WIPE — beats 1, 5: swell 1.65, blend 0.98. Dots scale up + blur into wash, then resolve. Most explosive.
DOT BLOOM — beat 2: swell 1.35, blend 0.85. Radius swells, raster merges briefly, hue + sat kick.
SOFT CUT — beat 3: swell 0.35, blend 0.2. Same field keeps moving, subtle lateral drift.
CURRENT — beat 4: slow domain warp, restrained. Hue tracks the magnifying glass.
PROOF BEAT — beats 6–10: measured lateral motion (±34px), precise, background serves the UI.
CUT-CURVE V2 60% — beat 11: bg accelerates with fg, field internals drag laterally, bloom at cut frame, decelerates into scene-00.
Spring Physics
Logo fly-through: high stiffness (180), moderate damping (14), ~15% overshoot. Panel slide-in: softer spring with slight overshoot. Magnifying glass entry/exit: spring in, gravity drop out. GSAP ticker-driven for logo, CustomEase approximations elsewhere.
Transition Protocol
Cut-the-curve for the final exit (beat 11). Same direction both sides (leftward). Exit: power3.in (42% of travel). Entry: expo.out (58%, continues same vector). Bloom peaks at cut frame — radius swell + color saturation + motion blur (6px). Field-space drag: mass positions, ribbon, center, phase all slide laterally with the plate. Background leads by 0.1s. Handoff to existing scene-00 must be seamless — same halftone grammar, same field.
